Observatorio de Precios Paraguay 3.0

Panel ejecutivo con mapa real, monitoreo por ciudades, índice semanal automático, ranking de productos, alertas de presión inflacionaria y estructura lista para publicar online. Esta versión ya está pensada para una presentación institucional seria.

Importación Excel .xlsx Mapa real con Leaflet Índice nacional semanal Ranking automático Exportación JSON y CSV Base local o demo online

Acciones rápidas

Primero abre el archivo en el navegador. Luego pulsa Cargar demo para ver el panel lleno con datos. Después ya puedes publicar el mismo archivo en GitHub Pages o Netlify.

Cómo dejarlo online correctamente

1. Guardar el archivo
Guarda este contenido como index.html en una carpeta nueva. Usa UTF-8 y Todos los archivos.
2. Probar localmente
Haz doble clic en el archivo. Verifica el mapa real, el gráfico y el botón Cargar demo.
3. Publicarlo online
Sube el archivo a GitHub Pages, Netlify o Railway Static. En esta versión ya no dependes de un mapa dibujado manualmente.
Formato recomendado para importar Excel
Usa una hoja con estas columnas exactas: Semana, Ciudad, Producto, Precio. Ejemplo: 2026-W12 | Asunción | Tomate | 13200.
Guía simple para subirlo a GitHub Pages
1) Crea una cuenta en GitHub. 2) Crea un repositorio nuevo, por ejemplo observatorio-precios. 3) Sube el archivo index.html. 4) En Settings → Pages, activa la publicación desde la rama main y carpeta /root. 5) GitHub te dará el enlace público.
